yf_tags.class.php

	Модуль предназначен для отбражения тегов в результатах других модулей (блог, галерея, статьи), а также
	осуществления поиска по тегам

	Основные методы:

	- show - метод по умолчанию - отображает облако тегов, форму для поиска.
	- _show - отображает блок с тегами для данного объекта (поста в блоге, фото в галерее, статьи и пр.).
	Подключается через модуль "yf_module"
	- _edit_tags - позволяет редактировать теги для данного объекта. Подключается через модуль "yf_module"
	- search - позволяет выполнять поиск посредством клика на имени тега в облаке, блоке с тегами и формы
	- tag - то же что и "search"
	- settings - позволяет редактировать настройки для каждого пользователя
	- _check_rights - определяет права на редактирование тегов для данного объекта. Внутренний метод

	function _show:

		Передаваемые параметры: 
		- $obj_id - id(s) объекта для которого отображается блок тегов (type:"array" or "int")
		- $params - массив дополнительных параметров (object_name - имя модуля для объекта которого
	отображается блок тегов ("blog","gallery","articles"))

		Возвращаемый результат: блок тегов содержащий теги со ссылками на поиск, либо надпись "No tags"
	при их отсутствии, стрелку, при нажатии на которую появляется блок ссылок с уточняющими действиями 
	и ссылку на редактирование тегов (если разрешено текущему пользователю).


	Теги подключаются следующим образом:
	- в массив разрешенных модулей "$this->avail_objects" (в функции "_init") необходимо добавить название
	модуля, в котором подключаются теги
	- вызвать функцию "$this->_show_tags()" в которую передать массив либо значение id объектов для которых
	показываются теги (поста в блоге, фото в галерее, статьи и пр.)
	Функция вернет массив собранных и готовых к показу блоков тегов ключами в котором будут являться id 
	переданных объектов
	- передать блоки тегов в шаблон объекта.
