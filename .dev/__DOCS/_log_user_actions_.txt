Хиты - все просмотры ваших записей, с учетом просмотра во френд-ленте и сообществах. Важно! Если ваш пост 
содержит lj-cut, то при просмотре поста засчитывается столько хитов, сколько раз был просмотрен пост в 
любом виде (один человек может прочитать анонс поста, потом кликнуть <читать далее>, 
просмотреть полный текст, а потом откомментировать пост, в этом случае будет засчитано ТРИ хита). 

Посетители - уникальные посетители вашего журнала в разных источниках (с учетом френд-ленты и сообществ). 
Уникальность определяется по cookies. Если один человек читал вашу запись во френд-ленте и потом в журнале, 
то будет засчитан один уникальный посетитель. 

Читатели - это посетители, который в момент прочтения вашей записи были авторизованы в ЖЖ, т.е. это 
подмножество посетителей. Уникальность читателя определяется по его нику в ЖЖ.

Количество полученных комментариев - показывает количество оставленных комментариев к вашим постам за 
указанным промежуток времени. Если комментарий был удален, то количество удаленных комментариев будет вычтено 
из числа оставленных.

История друзей - показывает историю прироста и уменьшения количества друзей во времени. При нажатии на число 
количества добавленных или удаленных друзей, вы можете увидеть имена пользователей.

// Stats // 
Мои гости - показывает не более 100 последних авторизованных пользователей, которые зашли на любую страницу 
журнала за последние 30 дней. Не учитываются пользователи, которые читали журнал через свою френд-ленту. 
Статистика обновляется раз в час.

Статистика записей - показывает прочтения по одной, выбранной вами записи в журнале. Отчет показывает 
раздельно статистику по ВСЕМ хитам и посетителям (включая френд-ленту и сообщества), и отдельно статистику по 
хитам и посетителям <в журнале>, т.е. прочтения на страницах дневника.
