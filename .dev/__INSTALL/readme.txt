Описание установки стандартного сайта основанного на фреймворке.

1. Создать папку для сайта. Например: "D:\www\htdocs\new\"
2. Скопировать в созданную папку файл "install.php"
3. Запустить инсталяцию. Например так: "http://localhost/install.php"
4. Далее следовать указаниям инсталлятора.


По умолчанию создаeтся:

	Тестовый пользователь:
	login: test
	password: test
